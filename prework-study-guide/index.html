HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • There should only be one h1 heading per page.
  • Always include alt text for images for accessibility and SEO.

CSS

  • CSS = Cascading Style Sheets. It makes your page look pretty.
  • Inheritance features can be added to the overall page instead of individual sections.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • "git" precedes every git statement.
  • Pull request: asking for your changes on your branch to be merged into another branch.
